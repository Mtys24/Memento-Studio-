# Memento Studio — Galería Digital Narrativa 🎨📐

Memento Studio es una plataforma web premium diseñada para fusionar el arte fotográfico y la artesanía textil en una experiencia digital sofisticada. Desarrollada con **React v18**, **Tailwind CSS v4** y **Framer Motion**, esta aplicación ofrece una narrativa visual inmersiva para el mercado de arte y decoración en Argentina.

## ✨ Características Premium
- **Esencia Dual (Theming Contextual)**: Cambio radical de estética entre los mundos de "Cuadros" (estilo museo) y "Tejidos" (estilo atelier).
- **Backend Invisible (Google Sheets)**: Sincronización en tiempo real con una hoja de cálculo para gestionar catálogos, precios y disponibilidad sin necesidad de un CMS complejo.
- **Narrativa Animada**: Transiciones cinemáticas y micro-interacciones que reflejan el trasfondo de diseño gráfico de la fundadora.
- **Modales de Alta Gama**: Visualización de producto con carruseles fluidos y llamadas a la acción directas a Instagram.

## 🛠️ Stack Tecnológico
- **Core**: React.js + Vite
- **Estilos**: Tailwind CSS v4 (Custom theme system)
- **Animaciones**: Framer Motion (Orchestration & Staggering)
- **Datos**: PapaParse (CSV Parsing) + Google Sheets API
- **Iconografía**: Lucide React

## 🚀 Instalación y Desarrollo

1.  **Clonar el repositorio**
    ```bash
    git clone <url-del-repo>
    cd memento-studio
    ```

2.  **Instalar dependencias**
    ```bash
    npm install
    ```

3.  **Iniciar servidor de desarrollo**
    ```bash
    npm run dev
    ```
